## 2.眼球血管偵測分析(U-Net模型)
   - 眼底鏡輸出的圖像經由openCV影像前處理,再由U-Net模型輸出眼球血管影像

## 3.新冠肺炎X光分類(CNN 影像辨識 + 遷移學習優化(以資料擴增優化創建 EfficientNet 模型))
   - 將肺炎X光片資料,使用 EfficientNet 模型進行Transfer Learning,進行輸出預測"正常","肺炎"或"新冠肺炎"

## 4.辨識腦腫瘤組織MRI影像(CNN 語意切割、萃取影像特徵=>U-Net模型)
